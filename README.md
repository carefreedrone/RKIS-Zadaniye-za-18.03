# Каталог интерьерных светильников

## О проекте
Учебный проект страницы каталога товаров с фильтрацией. 
Тема: интерьерные светильники в стиле минимализм.

## Ссылка на прототип
![Прототип страницы каталога](Прототип.png)

## Функционал
- Фильтрация по категориям, цене, брендам и наличию
- Сортировка товаров
- Переключение между плиткой и списком
- Пагинация и кнопка "Загрузить ещё"
- Интерактивная кнопка лайка

## Технологии
- HTML5
- CSS3 (Bootstrap 5 + кастомные стили)
- JavaScript (ES6+)

## Как запустить
1. Склонировать репозиторий
2. Открыть index.html в браузере
